# v2ray-openshift ddy
A v2ray docker image for openshift v3.
- work with websocket
- proxy_pass by nginx
- custom client_id and encryption method
- custom client_alterid
- add environment variables to edit optional setting
  - CLIENT_ID
  - CLIENT_ALTERID
  - CLIENT_SECURITY
- don't need custom domain and ssl certificate
- only cost 1 pods
